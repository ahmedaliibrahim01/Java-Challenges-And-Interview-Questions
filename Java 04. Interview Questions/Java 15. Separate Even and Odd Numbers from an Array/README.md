## Separate Even and Odd Numbers from an Array

### Problem Description
Create a Java program that separates even and odd numbers from an array. Make two lists evenNumbers and oddNumbers and add respective numbers in the lists.

### Example
    Test Input
    1
    2
    3
    4
    5
    6
---
    Expected Output
    [2, 4, 6]
    [1, 3, 5]
